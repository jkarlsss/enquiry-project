# AI Client Enquiry Processor

This is a small prototype built for Strata Management Consultants to help staff process incoming client enquiries more efficiently.

## Features

- Accepts a client enquiry as text input
- Uses Gemini AI to analyse the enquiry
- Classifies the enquiry type
- Generates a confidence score
- Summarises the enquiry
- Suggests a staff response
- Recommends the next action
- Flags whether human review is required
- Handles vague or invalid input

## Enquiry Categories

The AI can classify enquiries as:

- New client
- Support request
- Complaint
- General question
- Urgent issue
- Unclear
